Welcome to Project Unison

Project Unison is the project of community-made servers set in Team Fortress/Half-Life Universe aiming to be alternatives of Valve's Team Fortress 2 and Blizzard's Overwatch 2.

This server uses New-Old Weapons Pack Mod by SilentFox3. It is recommended to manually install the New-Old Weapons Pack mod for best experience.

Things to follow

1. Pornographic, gory, death-depicting, or any such abusive contents (e.g. sprays, profile pictures, nicknames) are not allowed.

2. Hateful, bigoted slurs, racism, doxxing, harassment or any words that could offend other mercenaries are not allowed.

3. Impersonation of other players, including staff members are discouraged.

4. Advertising other communities that are not affilated with Project Unison are not allowed. Simply posting a link for reference is okay!

5. Do NOT post URL shorteners/malicious links that would lead other mercenaries at risk.

6. Cheating, using third party mod that gives unfair advantage, or exploiting server's system are not allowed.
