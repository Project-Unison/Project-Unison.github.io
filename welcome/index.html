Welcome to Project Unison

Project Unison is the community-made dedicated server project aiming to be better alternative of PvP/PvE games, based on Asia.

Things to remember

1. Use common sense

2. Use in-game tools accordingly to mute players/disable sprays

3. Play fair; Don't cheat, crash servers or exploit things that would disturb other players' experiences

4. Have fun!
