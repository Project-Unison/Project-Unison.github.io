Welcome to Project Unison

Project Unison is the project of community-made servers set in Team Fortress/Half-Life Universe aiming to be alternatives of Valve's Team Fortress 2 and Blizzard's Overwatch 2.

This server uses New-Old Weapons Pack Mod by SilentFox3. It is recommended to manually install the New-Old Weapons Pack mod for best experience.

Things to follow

1. Pornographic, gory, death-depicting, or any such abusive contents (e.g. sprays, profile pictures, nicknames) are NOT allowed.

2. Hateful, bigoted slurs, racism, doxxing, harassment or any words that could offend other mercenaries are NOT allowed.

3. Impersonation of other players, including staff members are NOT allowed.

4. Advertising other communities that are not affilated with Project Unison are NOT allowed.

6. Cheating, using third party mod that gives unfair advantage, or exploiting server's system are not allowed.

List of commands

Click me to see the full list of commands!
Command Aliases Description Note
!nextmap - Shows the next map. -
!timeleft - Displays the remaining time of the map. -
!rtv rtv, !rockthevote Rock the vote to change the current map. -
!nominate - Nominate a map you want to play. Nominated maps will be displayed on next map vote. -
!revote - Re-opens the vote menu. Useful if you did some mistake when choosing during votes. This command only works during vote
!hop !server Shows the list of our server. -
!stuck - Unstucks yourself. Useful if you got stuck due to map objects. -
!call !calladmin Use this to alert the server admin about breaking rules. Abusing this command may result in a BAN
!votealltalk - Initiate the alltalk vote. Alltalk is off by default. This command can be used at any time
!votekick - Initiate the vote to kick a player who is trolling or cheating. Excessively abusing this command may result in a BAN
This command only works on 4 or more players on current server to prevent abuse
!votemenu - Opens the votemenu. -
