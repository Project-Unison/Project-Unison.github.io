Welcome to Project Unison Re:LIVE

Project Unison Re:LIVE is the community-made dedicated co-op server project aiming to achieve best PvE experience that others failed to do so.

Things to remember

1. Use common sense

2. Play fair; Don't cheat, crash servers or exploit things that would disturb other players' experiences

3. Have fun!

List of commands

Click me to see the full list of commands!
Command Aliases Description Note
!translator - Opens the translation menu which allows you to translate your native language to others. If you can't speak English, enabling this feature is highly recommended. -
!nextmap - Shows the next map. -
!nominate - Nominate a map you want to play. Nominated maps will be displayed on next map vote. -
!rtv rtv, !rockthevote Rock the vote to change the current map. -
!revote - Re-opens the vote menu. Useful if you did some mistake when choosing during votes. This command only works during vote
!stuck - Unstucks yourself. Useful if you got stuck due to map objects. -
!call !calladmin Use this to report players breaking the server rules. Abusing this command may result in a BAN
!voteslay - Initiate the vote to slay a player who is abusing exploits. Excessively abusing this command may result in a BAN
This command only works on 4 or more players on current server to prevent abuse
