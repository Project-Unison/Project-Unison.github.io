Welcome to Project Unison

Project Unison is the project of community-made servers set in Team Fortress/Half-Life Universe aiming to be alternatives of Valve's Team Fortress 2 and Blizzard's Overwatch 2.

This server uses New-Old Weapons Pack Mod by SilentFox3. It is recommended to download the New-Old Weapons Pack mod for best experience.

Server Rules (A.K.A Code of Conduct)

1. Be nice and use common sense. Any hate speech, racism, bigoted slurs, threats, harassment, spreading false rumors, or any offensive behaviors will NOT be tolerated, even it is a joke.

2. We are not here to post NSFW contents. Any Pornographic, gory, death-depicting, or any such abusive contents (e.g. sprays, profile pictures, nicknames) are NOT allowed.

3. Only safe links. Posting malicious/shortened links will NOT be tolerated.

4. Play fair. Cheating, using third party mod that gives unfair advantage, or exploiting server's system are not allowed.

5. If you are restricted for breaking rules, just keep calm and carry on. Any attempt to evade bans, mutes, gags will NOT be tolerated.
